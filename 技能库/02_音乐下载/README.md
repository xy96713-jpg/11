# 🎵 音乐下载 (Music Download Expert)

本模块为您提供纯净、无 YouTube 杂音的高品质音乐获取方案。

## ✨ 核心功能
*   **多源获取**：深度集成网易云、SoundCloud 等官方音源。
*   **智能元数据**：下载后自动写入 ID3 标签、封面图及专辑信息。
*   **批量处理**：支持通过歌单链接或文本列表进行大批量任务分发。

## 📖 操作指南
- **单曲下载**：直接发送歌曲名，并对我说：“**下载这个的高品质原声**”。
- **歌单同步**：发送网易云/SoundCloud 链接，并说：“**全量同步该歌单**”。
- **状态确认**：在 `assets/` 目录中可以查看下载历史及日志。

## 🛠️ 技术信息
- **版本**: V3.0 (Strictly non-YouTube)
- **更新时间**: 2026-01-31
- **核心组件**: 已集成 `NeteaseExpert` 与 `MusicDownloadExpert` 的全量逻辑。
