## ❓ 常见问题 (FAQ)

### Q1: 为什么有时候技能没激活？
**答**：可能是技能文件放错了位置（如系统隐藏目录）。
**解决**：确保技能文件夹都在当前的 `d:\anti\技能库\` 下。我在工作时只会扫描“眼前”的工具箱。

### Q2: 需要手动输入 `import` 吗？
**答**：**不需要！** 我具备“意图识别”能力。例如，当您提到“下载音乐”时，我会自动让 `02_音乐下载` 技能出列。

### Q3: 如何寻找方案？
**答**：使用 **Skill 06**。对我说 "Solve: 如何实现..."，我会自动进行全网搜索、技术对比，最后给您一个最佳实施方案。
