# 🛠️ 系统工具 (System Tools)

本模块为您提供 Chrome 及其周边环境的底层增强与维护能力。

## ✨ 核心功能
*   **Chrome AI 全量解锁**：激活 Glic 侧边栏、Prompt API 及 Gemini 离线模型。
*   **环境地理伪装**：一键锁定 US 区域，解决因 Location 导致的 API 封禁。
*   **进程与缓存管理**：自动化清理浏览器垃圾，优化 AI 模型的加载速度。
*   **诊断与审计**：提供可视化的 Chrome 系统健康报告。

## 📖 操作指南
- **一键解封**：直接对我说：“**帮我全量解锁 Chrome 的 AI 特性**”。
- **环境迁移**：说：“**把我的浏览器环境模拟为美国**”。
- **系统审计**：用于排查为什么 `window.ai` 无法调用的底层原因。
