# 🧠 深度研究 (Deep Research)

本模块旨在为您提供全网最强的信息搜集与汇总能力，突破地理封锁。

## ✨ 核心功能
*   **全网深度冲浪**：自动阅读网页、GitHub 仓库及行业报告。
*   **跨区域访问**：利用专有代理隧道绕过各种 Location 限制。
*   **知识自动合成**：将碎片信息整理成结构化的研究报告。

## 📖 操作指南
- **状态确认**：点开“📊 技能状态看板”确认模型是否加载。
- **调用方式**：您在对话中直接说：“**深度调研一下[某课题]**”或“**用 DeepResearch 分析[某链接]**”。
