# 🎧 DJ 智能助手 (DJ Intelligence)

本模块集成了最顶级的音频分析与 DJ 演奏辅助逻辑。

## ✨ 核心功能
*   **Rekordbox 深度同步**：解析 XML 数据库，同步标签与 Playlists。
*   **智能 Cue 点生成**：基于 AI 的乐句识别，全自动标注 4/8/16/32 小节 HotCues。
*   **能量与谐度分析**：评估 Mashup 和谐度（Key Alignment）与能量流行曲线。
*   **智能歌单策展**：自动生成符合现场能量流的曲目排序建议。

## 📖 操作指南
- **生成 Cue 点**：发送音频文件，并说：“**为此音轨生成 AI HotCues**”。
- **混音分析**：提供两首歌曲名，问：“**这两首歌 Mashup 的和谐度是多少？**”
- **环境审计**：直接说：“**审计我的 Rekordbox 数据库一致性**”。
