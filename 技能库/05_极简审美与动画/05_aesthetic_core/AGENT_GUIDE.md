# Skill: 极简审美与动画专家 (Aesthetic & Animations V2.1)

## 0. 核心定位
本 Skill 负责生成具有“高级感、流动感”的 UI 动效、粒子系统及交互组件。它是 Antigravity 系统的**视觉中枢**。

---

## 1. 核心流程 (Workflow)

### A. 灵感采集与推演
- 识别用户对“赛博朋克”、“极简”、“高级感”的关键词需求。
- 自动选择最合适的技术栈：GSAP, Particles.js, CSS3 keyframes, 或 Three.js。

### B. VQA 标准验证 (强制性)
- **见图如见面**：生成完代码后，必须调用 `browser_subagent`。
- **验证项**：首屏黑屏检查、交互滑块有效性、音频驱动反馈。
- **详见**：[VQA_PROTOCOL.md](./VQA_PROTOCOL.md)

### C. 画廊同步
- 自动将生成的 HTML 归档至 `gallery/` 目录。
- 更新本地画廊索引，确保用户可通过 `⚡_启动本地画廊.bat` 查看。

---

## 2. 操作规范 (Constraints)
1. **零外部依赖**：优先使用 CDN 或将资产 Base64 嵌入，确保单文件可离线打开。
2. **审美先行**：严禁给出默认的红蓝配方案。必须使用 HSL 调色盘，配合磨砂玻璃 (Blur) 与微动效。
3. **证据链条**：在交付前，Agent 必须确信自己已在浏览器中“亲手”调节过参数。
