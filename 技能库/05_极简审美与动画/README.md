# 🎨 极简审美与动画 (Aesthetic & Animations Expert)

本模块是您环境中的“视觉灵魂”，专门负责生成极具高级感、流动感且现代化的 Web UI 特效与动画组件。

## 🌟 核心能力

- **极致审美 UI 生成**：基于 Glassmorphism、Neomorphism 及现代暗色系美学，生成一秒惊艳的交互界面。
- **动态特效库**：包含粒子系统（Particles.js）、GSAP 补间动画及自定义 CSS3 动画序列。
- **品牌感塑造**：自动生成符合“Antigravity (反重力)”品牌调性的光效、渐变与呼吸感动效。

---

## 📖 用户操作指南 (User Manual)

### 1. 怎么生成特定的动画？
您可以描述一种氛围，或者直接要求特定技术的动效：
- **示例**：`“帮我做一个带‘赛博朋克’流动感的登录界面”`
- **示例**：`“为一个音乐播放器设计一套带呼吸灯效果的 CSS 动效”`
- **示例**：`“生成一个极简、高级的粒子背景动画，要带流动感”`

### 2. 怎么获取昨日的那些动画？
所有昨日生成的网页动效模板均已归档。您可以问：
- **示例**：`“展示一下昨天那个‘反重力脉冲’的动效代码”`
