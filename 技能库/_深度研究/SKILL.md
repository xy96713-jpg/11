# 🧠 深度研究智能 (Deep Research Intelligence)

## 📌 概述
本技能专为处理长周期、多维度的信息搜集任务而设计。它整合了自主浏览器代理与云端长上下文处理能力，能够跨越地理限制调取全网信息。

## 🛠️ 核心组件
- `main.py`: (即将生成) 统一调用入口。
- `ultra_browser_agent.py`: 基于 Playwright 的无头浏览器控制。
- `final_ai_handshake.py`: 用于验证本地环境与 API 的挂载。

## 📖 使用场景
1. **替代 NotebookLM**：当官方服务因区域封锁不可用时。
2. **多源校对**：针对同一课题对比 GitHub, 社交媒体与学术论文的不同观点。
3. **技术路线调研**：为开发者提供高可用的代码实例与架构建议。
