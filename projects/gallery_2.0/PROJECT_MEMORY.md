# Project Memory: Studio Gallery 2.0

## 🎯 任务目标
重构画廊为“成品-草稿”双级系统，并集成 PM 协议与视觉智脑分析引擎。

## 🧠 技术选型与决策
- **分类策略**：强制精简，只允许“Finished”和“Drafts”。
- **交互规范**：采用全屏 Viewer 系统，配备 Glassmorphism 调节面板。
- **分析引擎**：基于 HSL 色彩空间与物理动效模型。
