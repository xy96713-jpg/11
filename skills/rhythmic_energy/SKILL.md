# Rhythmic Energy (律动能核)

## 0. 核心能力
本技能专注于 Set 的动态流程控制，确保混音过程中的能量衔接平滑，避免律动冲突。

## 1. 技术组件
- **phrase**: 分析曲目的乐句结构，定位 Mix-in 和 Mix-out。
- **bpm**: 处理 BPM 的平滑跃迁。

## 2. 使用场景
- 需要在 Set 中进行跨速度 (BPM Jump) 排列时。
- 确定具体的切歌长度和点位时。
